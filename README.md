# Bouzuya

bouzuya's gem.

## Installation

Add this line to your application's Gemfile:

    gem 'bouzuya'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bouzuya

## Usage

```
require 'bouzuya'

p Bouzuya::homepage #=> "http://bouzuya.net/"
p Bouzuya::blog #=> "http://blog.bouzuya.net/"
```
